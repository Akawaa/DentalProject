Dentalproject

Projet de stage de DUT Informatique

View on GitHub Download .zip Download .tar.gz

Contexte

Projet réalisé à l'Université de Portsmouth en Angleterre durant mon stage de fin d'année de DUT Informatique.

Sujet

Je devais améliorer un site Web pour un cabinet dentaire fictif, nommé South Sea Dental. Le site doit permettre la gestion d'un cabinet dentaire, c'est-à-dire, prise de rendez-vous, fiche patient avec les soins réalisés, les radios, les factures, le prochain rendez-vous, etc. Les patients peuvent également se connecter sur un compte personnel afin de voir la date de leur futur rendez-vous, mais également avoir accès à toutes ses factures, ses soins, ses anciens rendez-vous.

Technologie

J'ai fais la migration du site Web, initialement développé en PHP procédural, vers le framework CodeIgniter utilisant le paradigme MVC. Pour le design, j'ai utilisé le framework Bootstrap. La base de donnée est gérée sous mySQL5.5. J'ai également utilisé du Javascript, notamment Ajax.

Installation

  1. Télécharger le dossier.
  2. Extraire l'archive dans votre serveur local.
  3. Créer et importer la base de données. Fichier DentalProject.sql
  4. Modifier le fichier config.php situé dans applications>config : ligne 20 : $config['base_url'] = 'http://your-url/DentalProject/';
  5. Modifier le fichier database.php situé dans application>config : ligne 68 : 'username' => '', ligne 69 : 'password' => '', ligne 70 : 'database' => '',
